How It Works

This application uses event sourcing to track all changes to bank accounts:

  1. Every action (deposit, withdrawal, transfer) creates an event
  2. Events are stored in SQLite and never deleted
  3. Account state is rebuilt by replaying events
  4. Full audit trail is maintained automatically
