A demo application showcasing:

  • Evento - Event sourcing library
  • Evento-ACCORD - Distributed consensus protocol
  • Axum - Web framework

Create Your First Account View Accounts

How It Works

This application runs on a 3-node ACCORD cluster:

  1. Each node has its own SQLite database
  2. All writes go through ACCORD consensus
  3. Transactions are replicated to all nodes
  4. You can write to any node and read from any node

Try creating an account on one node and viewing it on another!
